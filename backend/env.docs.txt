Environment Variables Documentation
Database Configuration

MONGODB_URI - MongoDB database connection string
REDIS_HOST - Redis server hostname/IP address
REDIS_PORT - Redis server port number

Server Configuration

CLIENT_SERVER - Client/frontend server URL
PORT - Application server port number
NODE_ENV - Node.js environment (development/production/test)

Email/SMTP Configuration

SMTP_HOST - SMTP server hostname
SMTP_PORT - SMTP server port number
SMTP_USER - SMTP authentication username
SMTP_PASS - SMTP authentication password
SMTP_FROM - Default sender email address

Authentication & Security

JWT_SECRET - Secret key for JSON Web Token signing
REFRESH_TOKEN_SECRET - Secret key for refresh token signing
SESSION_SECRET - Secret key for session management

Cloud Storage (Cloudinary)

CLOUDINARY_CLOUD_NAME - Cloudinary cloud name
CLOUDINARY_API_KEY - Cloudinary API key
CLOUDINARY_API_SECRET - Cloudinary API secret

Payment Processing (Stripe)

STRIPE_KEY - Stripe API key (publishable or secret)
STRIPE_WEBHOOK_SECRET - Stripe webhook endpoint secret
STRIPE_URI - Stripe-related URI/endpoint

Video Communication (ZEGO)

ZEGO_APP_ID - ZEGO Cloud application ID
ZEGO_SERVER_SECRET - ZEGO server secret key
ZEGO_SERVER - ZEGO server URL/endpoint
ZEGO_APP_SIGN - ZEGO application signature/token
